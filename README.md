# Digital Grimoire

This repository is a living knowledge system.

It is not documentation.
It is not a notebook.
It is not a second brain.

It is a **Grimoire**:
- knowledge with intent
- systems that evolve
- tools that are summoned when needed

This grimoire is designed to be:
- personal first
- extensible to multi-user
- compatible with AI agents
- auditable through time
